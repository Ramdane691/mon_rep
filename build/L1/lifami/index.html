Introduction à Grapic

Pour utiliser Grapic, il faut inclure la bibliothèque, ouvrir une fenêtre graphique, puis exécuter une boucle d’affichage : 

#include <Grapic.h>
using namespace grapic;

int main()
{
    winInit("Fenêtre Grapic", 800, 600);

    while (!winHasFinished())
    {
        backgroundColor(255, 255, 255);
        clear();

        // Dessins ici
        color(0, 0, 255);
        drawLine(100, 100, 200, 200);

        winDisplay();
    }

    return 0;
}

Exemples de TP

Voici quelques exemples de projets réalisés dans cette UE :

  • Création d’un labyrinthe, d’un pong ou d’une animation

Exemple : affichage d'une parabole y = ax²

for (float x = -10; x < 10; x += 0.1)
{
    float y = a*x*x;
    drawLine(400 + x*20, 300 + y*20, 400 + x*20, 300 + y*20);
}

Mathématiques appliquées

  • Géométrie (repère, coordonnées, vecteurs)
  • Suites numériques, fonctions (affichées graphiquement)
  • Calculs vectoriels (norme, direction, rebond)
  • Fractales et récursivité (ex : arbre, flocon de Koch...)

Exemple : Arbre récursif

Voici un exemple typique : un arbre dessiné récursivement avec Grapic.

void arbre(float x, float y, float angle, float taille, int n)
{
    if (n == 0) return;

    float x2 = x + taille * cos(angle);
    float y2 = y + taille * sin(angle);

    drawLine(x, y, x2, y2);

    arbre(x2, y2, angle - 0.3, taille * 0.7, n - 1);
    arbre(x2, y2, angle + 0.3, taille * 0.7, n - 1);
}

Appel initial dans la boucle principale :

arbre(400, 100, M_PI/2, 100, 7);

Conclusion

Cette UE te montre comment passer d’un problème mathématique abstrait à un programme fonctionnel en C++, et parfois même à une visualisation graphique. Tu apprendras à mieux structurer tes fonctions, utiliser des boucles et de la récursivité, tout en manipulant des objets visuels simples avec Grapic.
